## Installation
> ### Clone the repo.
>>   ```bash
>>  # Clone this repository
>>  $ git clone https://github.com/FelipePardim/Platform-Tower-Defense
>>   ```
> ### Open the project folder on godot project manager.
>
> ### Feel free to run, study and modify the project.
